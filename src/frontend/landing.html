Self-Host in 3 Steps

git clone https://github.com/eddielathamjones/geochron-web.git cd geochron-web pip install -r requirements.txt python -m src.backend.app
# Or with Docker: docker build -t geochron-web . docker run -p 5002:5002 geochron-web

Then embed it: <iframe src="http://your-server/embed?style=dark" width="100%" height="400"></iframe>
